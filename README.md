# Fireball UI-Kit
The Fireball UI Kit used in Editor Framework.

Build for [`kss-node`](https://github.com/kss-node/kss-node).

----

# preview

[preview](http://w88975.github.io/ui-kit/bin/index.html)

----

# Install
```shell
# install kss-node
$ npm install -g kss

#install polymer
$ bower install .
```
----

# Build

```shell

# dist path: ./bin/
$ kss-node src/ bin/ --template template/

```
